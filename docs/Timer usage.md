# Timer usage

| Timer   | Usage                                 |
| ------- | ------------------------------------- |
| `TIM1`  | Unused                                |
| `TIM2`  | Custom delay                          |
| `TIM3`  | Left encoder(DC) or left step motor   |
| `TIM4`  | Right encoder(DC) or right step motor |
| `TIM5`  | Sensor interrupt                      |
| `TIM9`  | Speed control                         |
| `TIM10` | PWM                                   |
| `TIM11` | PWM                                   |

*Note that `TIM3`, `TIM4` are used for multiple purpose.*
